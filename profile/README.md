# Disfox

Disfox is a framework built to enhance the experience of working with **discord.js**.  
It introduces a cleaner and more structured way to organize your application code.

The goal is simple: keep things scalable, readable, and easy to maintain.

---

## 🌐 Website
<a href="https://disfox.netlify.app">https://disfox.netlify.app</a>

---

## 🧩 Purpose

- Improve project structure when using discord.js  
- Reduce repetitive code  
- Provide a solid foundation for scalable applications  
- Make development faster and more intuitive
